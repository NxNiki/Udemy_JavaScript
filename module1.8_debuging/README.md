# tutorial for chrome dev tools:

https://developer.chrome.com/docs/devtools/

## debug javascript:
https://developer.chrome.com/docs/devtools/javascript

open dev tools:
Linux:
```
ctrl + shift + j
```
or on Mac:
```
command + option + j
```

Press `Esc` to open the console.
